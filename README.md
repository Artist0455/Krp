# 🎶 Artist Music Bot

A simple Telegram Music Bot for voice chat.

## 🚀 Features
- Fast play, pause, resume, seek
- Now playing
- Artist check every 1 min
- Welcome & support/update channel

## 🛠️ Deployment
Set environment variables in Render:

- `BOT_TOKEN`
- `API_ID`
- `API_HASH`
- `OWNER_ID`
- `SUPPORT_CHAT`
- `UPDATE_CHANNEL`
- `ARTIST_CHECK_CHAT`

Then deploy with `worker: python3 main.py`
